# Iron Man Mark LXXXV Interactive Showcase

## Overview

This project brings the iconic Iron Man Mark LXXXV suit to life right in your browser. It's an immersive experience that lets you interact with a dynamic display of the suit's systems and memorable moments, making you feel like you're diving into a high-tech interface.

## Description

Ever wanted to see the Iron Man suit boot up with J.A.R.V.I.S. at your command? This interactive showcase lets you do just that. It's a visually rich, engaging experience designed to emulate Tony Stark's advanced tech interfaces. You'll scroll through a cinematic sequence, reveal key dialogues, and explore the suit's "telemetry" as if you're Stark himself. It's all about delivering a polished, high-performance web experience that feels truly next-gen.

## Installation

Getting this project up and running on your local machine is straightforward.

1.  **Clone the Repository:**

    ```bash
    git clone https://github.com/NumCodes/iron-man.git
    cd iron-man
    ```

2.  **Install Dependencies:**
    Using npm:
    ```bash
    npm install
    ```
    Or using yarn:
    ```bash
    yarn install
    ```

3.  **Run the Development Server:**
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

## Usage

Once the application is running, it's designed for an interactive scrolling experience:

*   **Initial Load**: You'll see a loading sequence that builds the Mark LXXXV suit, complete with a progress indicator.
*   **Hero Section**: Scroll down to initiate the cinematic boot-up of the suit. As you scroll, frames of the suit animating will play, and key quotes from the Iron Man films will appear.
*   **Cinematic Reveal**: Continue scrolling to transition into a more dramatic reveal of the suit, accompanied by additional iconic quotes and dynamic UI elements.
*   **Systems Nominal**: Keep scrolling to reach the "Systems Nominal" section, where you can view simulated telemetry data for the suit and access the archive.

This project is best experienced by just scrolling through the entire page and enjoying the visual storytelling and interactive elements.

## Features

*   **Cinematic Scroll Animation**: Smooth, frame-by-frame animation triggered by user scroll, creating a dynamic visual narrative.
*   **Interactive UI Overlays**: HUD-like elements, progress bars, and data readouts that respond to scroll position.
*   **Dynamic Content Reveal**: Quotes and information cards appear and fade based on the scroll progress, adding context to the visual experience.
*   **Smooth Scrolling**: Utilizes Lenis for an enhanced, buttery-smooth scrolling experience across the entire site.
*   **Responsive Design**: Adapts beautifully to different screen sizes, from mobile devices to large desktop displays.
*   **Modern Component Architecture**: Built with reusable React components, ensuring maintainability and scalability.
*   **Performance Optimized**: Image sequence loading with progress feedback ensures a seamless experience even with heavy assets.
